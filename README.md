# Huffman Compression for Educational Resources

This project is a **web-based file compression system** that uses **Huffman Coding** to compress educational materials (PDF, TXT, HTML) for efficient storage and delivery in e-learning platforms.

## Features
- User registration and authentication
- File upload and Huffman-based compression
- Compression history with download and delete options
- Secure file-type validation (.pdf, .txt, .html)
- SQLite database integration

## Tools & Technologies
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Algorithm:** Huffman Coding
